# Focus Sessions

This folder is auto-populated by [[Focus/Focus Timer]]. Every time you stop & log a session, a new note appears here (organized into `YYYY-MM` subfolders), containing the project, task, start/end time, duration, and any notes.

You don't need to create anything here manually — but you can edit a session note directly (e.g. to fix a duration) if needed, since it's just a normal markdown note with frontmatter.
